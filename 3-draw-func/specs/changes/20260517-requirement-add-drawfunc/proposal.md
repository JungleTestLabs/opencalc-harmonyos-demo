# Proposal — 函数图像绘制（Draw Function Graph）

## 意图分类

| 维度 | 值 |
|------|-----|
| 意图类型 | `requirement-add`（新增功能） |
| 领域 | UI + Canvas 绘图 + 计算引擎复用 |
| 父模块 | 新增独立页面 `DrawFuncPage` |

## 需求描述

在 OpenCalc 基础上增加函数图像绘制功能：
- 用户输入数学表达式（如 `sin(x)`、`x^2`、`cos(x)*2`）
- Canvas 实时绘制函数曲线
- 显示坐标轴、网格线、刻度标记
- 支持 x 范围调整（默认 -2π 到 2π）
- 从导航页进入

## 影响层次

| 层级 | 影响 |
|------|------|
| UI 层 | 新增 DrawFuncPage.ets（~300 行） |
| 导航层 | 修改 Index.ets（+1 按钮） |
| 计算引擎 | **复用** CalcEngine.evaluate()，无修改 |
| 数据模型 | 无修改 |

## 复杂度评估

- **复杂度**: M 级（Canvas 绘图是新领域）
- **理由**: 涉及 CanvasRenderingContext2D API、坐标系转换、函数采样。虽然代码集中在一个文件，但 Canvas 绘图是独立技术领域。
- **需执行**: 完整 PLANNING 流程（含架构设计）

## 验收标准

1. 输入 `sin(x)` → 显示正弦曲线（-2π 到 2π，2 个完整周期）
2. 输入 `x^2` → 显示抛物线
3. 输入 `cos(x)` → 显示余弦曲线
4. Canvas 包含 x/y 轴、网格线、刻度标记
5. 从导航页点击进入，能返回主页
6. 空输入时不崩溃，显示提示
